Magnifier

This app was created for the visually impaired. The app allows the user to zoom into text, apply visual effects and toggle the flash in order to be able to read text. The app also supports OCR (Optical Character Recognition) which allows the user to use the camera to read text by showing boxes containing the words and reading it one by one.
